# Debugging CLI issues

If you're running into CLI errors, first try updating to the latest version of the CLI.

You can also add the `--verbose --log-level debug` options to output detailed logs in the CLI. The output will tell you whether errors are coming from a specific file — you can then use the `--ignore` option to filter out those files and complete the sync.
